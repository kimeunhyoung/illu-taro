# Illuminati Tarot Master (일루미나티 타로)

정적 웹앱: `index.html` + `data.js`

## 웹에서 보기

GitHub Pages를 켠 뒤 아래 주소로 접속합니다.

**https://kimeunhyoung.github.io/illu-taro/**

### Pages 처음 켤 때 (한 번만)

1. 저장소 **Settings** → **Pages**
2. **Build and deployment** → Source: **GitHub Actions** 선택
3. `master`에 푸시된 뒤 Actions 탭에서 워크플로가 성공하면 위 URL이 열립니다.

(이미 **Deploy from a branch**로 켜 두었다면 그대로 사용해도 됩니다.)

### 이 주소에 대해

- **검색에 광고되는 것은 아닙니다.** 주소를 본인만 북마크해 두고 쓰는 식으로 쓰면 됩니다.
- 다만 **링크를 아는 사람은 누구나** 열 수 있어요. 비밀번호로 막는 기능은 GitHub Pages 기본에는 없습니다. 링크를 공유하지 않으면 실질적으로 본인만 쓰는 용도로 쓸 수 있습니다.
